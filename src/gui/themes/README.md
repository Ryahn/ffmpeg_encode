# GUI theme assets

- **`video_encoder_dark.json`** — CustomTkinter color theme (loaded at startup via `gui.theme.apply_theme()`).
- **`app-theme.css`** (repo root) — design reference with the same palette; not parsed by the app.

When changing colors, update the JSON and `gui/theme.py` constants together, and adjust the CSS if you use it for mockups or docs.
